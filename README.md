# Kana Beat

A rhythm-based kana pronunciation prototype.

## PR 1 scope

- Vowels only: あ・い・う・え・お
- Fixed 80 BPM rhythm
- 20 prompts per round
- Japanese speech recognition through the Web Speech API
- Correct / wrong feedback
- Score, combo, accuracy, and final results
- Manual fallback buttons when speech recognition is unavailable

## Run locally

```bash
npm install
npm run dev
```

For microphone access, use a supported browser on `localhost` or HTTPS.
